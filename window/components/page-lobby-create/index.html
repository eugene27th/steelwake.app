РЕЖИМ ИГРЫ

ТЕХНИКА КОМАНДЫ
Вы выбираете технику, которая будет доступна в списке для выбора у игрока каждой команды.
Фильтры над списками помогут найти подходящую технику. Кнопка «выбрать всё» выбирает всю технику из списка, учитывая выбранные в данный момент фильтры.
